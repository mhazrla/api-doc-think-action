# Yearly Goal

As the name suggests, Yearly Goal are a core part of Protocol — the very reason Protocol exists is so you can have secure conversations with your Yearly Goal. On this page, we'll dive into the different yearly goal endpoints you can use to manage Yearly Goal programmatically. We'll look at how to query, create, update, and delete Yearly Goal.

## The Yearly Goal model

The Yearly Goal model contains all the information about your Yearly Goal, such as their username, avatar, and username. It also contains a reference to the conversation between you and the yearly goal and information about when they were last active on Protocol.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the Yearly Goal.
  </Property>
  <Property name="resolutions" type="string">
    The resolutions for the Yearly Goal.
  </Property>
  <Property name="photo" type="string">
    The photo for the Yearly Goal.
  </Property>
  <Property name="due_date" type="string">
    The Yearly Goal due_date in the Yearly Goal list.
  </Property>
  <Property name="share_with" type="string">
    The Yearly Goal share_with in the Yearly Goal list.
  </Property>
  <Property name="category_id" type="string">
    The Yearly Goal category_id in the Yearly Goal list.
  </Property>
  <Property name="user_id" type="boolean">
    The yearly goal accout in the Yearly Goal list.
  </Property>
</Properties>

## Create a Yearly Goal {{ tag: 'POST', label: '/yearly-goals' }}

  <Col>

    ### Required attributes

      </Property>
      <Property name="resolutions" type="string">
        The resolutions for the yearly goal.
      </Property>
      <Property name="photo" type="string">
        The photo for the yearly goal.
      </Property>
      <Property name="due_date" type="datetime">
        The yearly goal due_date for the yearly goal.
      </Property>
      <Property name="share_with" type="string">
        The yearly goal share with for the yearly goal.
      </Property>
      <Property name="category_id" type="string">
        The yearly goal category  for the yearly goal.
      </Property>
      <Property name="user_id" type="string">
        The yearly goal id for the yearly goal.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/yearly-goals">
